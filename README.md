# 🎬 Movie Sales Analysis (Power BI Project)

## 📌 Overview
This project analyzes movie data using Power BI to uncover insights about revenue, ratings, genres, and audience engagement. The dashboard helps in understanding trends and supports data-driven decision-making in the movie industry.

---

## 🎯 Objectives
- Analyze movie performance over time  
- Identify top-performing genres and directors  
- Understand revenue and rating trends  
- Build an interactive Power BI dashboard  

---

## 📁 Dataset Information
- **Source:** CSV files  
- **Time Period:** 2006 – 2016  
- **Data Includes:**
  - Movie Title  
  - Genre  
  - Release Year  
  - Runtime (Minutes)  
  - IMDb Rating  
  - Votes  
  - Metascore  
  - Revenue (Millions)  
  - Director / Actors  

---

## 🧹 Data Preparation
Data cleaning and transformation were performed using Power Query:

- Removed duplicate records  
- Handled missing values  
- Converted data types  
- Split genre column into multiple categories  
- Created calculated measures using DAX  

---

## 📊 Dashboard Features

### 📈 Movie Overview & Trends
- Movies released per year  
- Revenue trends over time  
- Average rating and runtime analysis  

### 🎭 Genre Analysis
- Genre distribution  
- Revenue by genre  
- Average rating and metascore  

### 🏆 Top Performers
- Top movies by revenue  
- Most voted movies  
- Top directors  

### 🔍 Advanced Insights
- Revenue vs Rating analysis  
- Runtime vs Rating  
- Genre-wise performance over time  

---

## 🛠 Tools & Technologies
- Power BI  
- Power Query  
- DAX (Data Analysis Expressions)  

---

## 📌 Key Insights
- Certain genres generate higher revenue  
- Ratings and votes influence movie success  
- Top directors consistently deliver high-performing movies  
- No single factor determines success — multiple factors contribute
